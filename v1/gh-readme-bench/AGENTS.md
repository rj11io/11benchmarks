# GitHub README benchmark runs

If you are given a prompt under `benchmark/prompts/`, it is the authoritative
task specification.

For a run:

- Write only inside the assigned `runs/<run-id>/` directory.
- Treat `ref/`, `benchmark/`, the repository README, and other run folders as
  read-only.
- Do not inspect or borrow from another run.
- Preserve the complete process in `research.md`, `content.md`, and
  `README.md`.
- Do not fabricate facts, sources, metrics, links, or live integrations.
